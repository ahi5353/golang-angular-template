# Golang Angular Template

このプロジェクトは、バックエンドに Go、フロントエンドに Angular を使用したテンプレートアプリケーションです。初期ユーザー登録、ログイン、ダッシュボードなどの基本的な認証機能が含まれています。

## 技術スタック

- **Backend**: Go
- **Frontend**: Angular
- **UI Framework**: Angular Material
- **Database**: SQLite

## 機能

- **初期ユーザー登録**: システムにユーザーが存在しない場合、自動的に登録ページにリダイレクトします。
- **認証**: ログインおよびログアウト機能。
- **ダッシュボード**: ログイン後にのみアクセス可能な保護されたエリア。

## プロジェクト構成

- `backend/`: Go アプリケーションコード
- `frontend/`: Angular アプリケーションコード

## 始め方

### 前提条件

- Go (1.20 以降)
- Node.js (LTS バージョン)
- Angular CLI

### バックエンドのセットアップ

1. backend ディレクトリに移動します:
   ```bash
   cd backend
   ```
2. 依存関係をインストールします:
   ```bash
   go mod download
   ```
3. サーバーを起動します:
   ```bash
   go run main.go
   ```
   バックエンドサーバーは `http://localhost:8080` で起動します。

### フロントエンドのセットアップ

1. frontend ディレクトリに移動します:
   ```bash
   cd frontend
   ```
2. 依存関係をインストールします:
   ```bash
   npm install
   ```
3. 開発サーバーを起動します:
   ```bash
   npm start
   ```
   アプリケーションは `http://localhost:4200` で利用可能になります。

## 使い方

1. ブラウザを開き、`http://localhost:4200` にアクセスします。
2. アプリを初めて実行する場合（データベースにユーザーがいない場合）、登録ページにリダイレクトされます。
3. 初期ユーザーを作成します。
4. 作成した認証情報でログインし、ダッシュボードにアクセスします。
